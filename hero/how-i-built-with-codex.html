How I Built Element Heroes with Codex

I started with a simple student idea during March Break: build a chemistry card game that makes science feel more fun, more visual, and easier to connect to real class learning. I did not begin with a full codebase. I began with an idea, and then used Codex to help me turn that idea into a real working project.

My Story

From an Idea to a Real Game

At first, I asked Codex to help me build the main game system. Together we created cards, reactions, attacks, HP, energy, practice mode, and multiplayer mode. Once the game started working, I kept testing it and asking for improvements.

After that, I wanted the project to be more educational, not just fun. I asked Codex to add science explanations, tutorials, teacher tools, and better wording for Grade 9 science. Later, I also asked it to fix places where the chemistry was too simplified or not accurate enough, such as treating Water and Carbon Dioxide as compounds instead of elements.

I also spent a lot of time on design and polish. I asked Codex to improve animations, make the game easier to use on phones, add stats, create an element voting page, and build special effects for cards like Fireball, Extinguish, and Corrode.

What I learned from this project is that AI coding does not replace my ideas. It helps me turn my ideas into something real faster. I still had to test, decide, improve, and keep asking for better versions, but Codex helped me build much more than I could have done alone.

Prompt Themes

Main Prompt Directions I Used

Game Build

Build the core game

  • Create a chemistry battle card game with reactions and strategy.
  • Add practice mode and multiplayer mode.
  • Use field, hand, HP, energy, and status systems.
Learning

Make it educational

  • Connect the game to Grade 9 science.
  • Add tutorials, science coach notes, and teacher-friendly features.
  • Fix wording so students do not learn incorrect chemistry.
Polish

Improve the experience

  • Add cinematic animations and better attack effects.
  • Improve the mobile card layout and text fit.
  • Fix bugs when effects or UI behaved incorrectly.
Growth

Add community features

  • Create live stats and a vote page for new elements.
  • Add Carbon Dioxide, Extinguish, and Fireproof.
  • Make the project strong enough to show in a club meeting.
