Grade 9 Data Management Review

Box Plots: Read Them, Build Them, and Compare Them

A box plot gives a fast picture of a data set using the five-number summary: minimum, lower quartile (Q1), median, upper quartile (Q3), and maximum. Use this page to review the core ideas and then try examples that get steadily harder.

Middle 50% The box from Q1 to Q3 shows where the middle half of the data lives.
Spread Longer whiskers or a wider box mean the data is more spread out.
Median The line inside the box marks the middle value of the ordered data.

Quick Skill Check

  1. Put the data in order from least to greatest.
  2. Find the median.
  3. Find Q1 from the lower half and Q3 from the upper half.
  4. Write the five-number summary.
  5. Sketch the box plot using a number line.
  6. When comparing box plots, talk about center, spread, and possible outliers.
Range Range = maximum - minimum. It shows the total spread.
Interquartile Range IQR = Q3 - Q1. It shows the spread of the middle 50%.
Symmetry clue If the median is centered in the box and whiskers look balanced, the data may be roughly symmetric.
Skew clue If one whisker is much longer, the data may stretch more in that direction.
Outlier rule A value may be an outlier if it is below Q1 - 1.5 × IQR or above Q3 + 1.5 × IQR.

Key Ideas

  • A box plot does not show every value. It summarizes the data.
  • The box starts at Q1 and ends at Q3.
  • The line inside the box is the median.
  • The whiskers extend to the minimum and maximum values unless outliers are handled separately.
  • If two data sets have the same median, they can still have very different spread.
  • To test for outliers, first find the IQR, then calculate the lower and upper fences.

Grade 9 questions often ask you to read a box plot, create one from a list of numbers, or compare two box plots using clear words like higher median, greater range, or more consistent.

Easy

Find the five-number summary

Here is a small ordered data set showing quiz scores out of 20.

4, 6, 7, 9, 10, 12, 13, 15, 18
Question: Find the minimum, Q1, median, Q3, and maximum.
Worked answer:
  1. The median is the middle value, so median = 10.
  2. The lower half is 4, 6, 7, 9, so Q1 is the middle of 6 and 7, which is 6.5.
  3. The upper half is 12, 13, 15, 18, so Q3 is the middle of 13 and 15, which is 14.
  4. Minimum = 4 and maximum = 18.

Five-number summary: 4, 6.5, 10, 14, 18

IQR: 14 - 6.5 = 7.5

Medium

Compare two box plots

Two classes wrote the same test. Use the summaries below to compare them.

Class A: min 48, Q1 56, median 64, Q3 72, max 88 Class B: min 40, Q1 60, median 66, Q3 70, max 74
Question: Which class had the higher typical mark, and which class was more consistent?
Worked answer:
  1. Look at the medians to compare typical marks. Class B has median 66, which is slightly higher than Class A's median 64.
  2. Look at spread to compare consistency. Class A has range 88 - 48 = 40, while Class B has range 74 - 40 = 34.
  3. Now compare the middle 50% using IQR. Class A has IQR 72 - 56 = 16. Class B has IQR 70 - 60 = 10.

Conclusion: Class B had the higher typical mark and was more consistent because it has a slightly higher median and a smaller IQR.

A common mistake is using only the range. The IQR is often better for discussing how tightly grouped the middle half of the data is.
Tricky

Outlier and skew thinking

A team recorded the number of minutes students spent on homework one night.

12, 14, 15, 16, 18, 18, 20, 22, 24, 55
Question: Find the five-number summary and explain what the box plot would suggest about the data.
Worked answer:
  1. The data is already ordered.
  2. There are 10 values, so the median is the average of the 5th and 6th values: (18 + 18) / 2 = 18.
  3. The lower half is 12, 14, 15, 16, 18, so Q1 = 15.
  4. The upper half is 18, 20, 22, 24, 55, so Q3 = 22.
  5. Minimum = 12 and maximum = 55.

Five-number summary: 12, 15, 18, 22, 55

IQR: 22 - 15 = 7

Outlier check: 1.5 × IQR = 10.5, so the fences are 15 - 10.5 = 4.5 and 22 + 10.5 = 32.5.

Decision: 55 is greater than 32.5, so 55 is an outlier.

Interpretation: Most homework times are packed between 12 and 24 minutes, but 55 is much larger than the rest. The right whisker would be much longer, so the data looks right-skewed and may contain an outlier.

This is tricky because the very large maximum can pull your attention away from the middle. A box plot helps you notice that most students are still clustered much lower.

Quick Quiz: Which Class Did Better?

Two classes wrote the same test. Look at the box plots and decide which class had the better overall scores. Then explain your thinking in one or two sentences.

How to judge
  1. Look at the median first. A higher median usually means a higher typical score.
  2. Look at the IQR next. A smaller IQR means the middle 50% is more consistent.
  3. Look at the range last. A very large range can mean the class had uneven results.
  4. Use words like higher median, smaller IQR, and more consistent in your answer.
Answer guide

Class B did better overall because its median is higher. It is also more consistent because its box is narrower, which means its IQR is smaller.

Class A

Median = 64 and the middle 50% is fairly spread out.

Class B

Median = 66 and the middle 50% is packed more tightly together.

What To Say In Your Answer

When a teacher asks you to compare box plots, write short evidence-based sentences instead of only naming numbers.

Sentence starters
  • The median of ___ is higher, so it has the higher typical value.
  • The IQR of ___ is smaller, so its middle 50% is more consistent.
  • The range of ___ is larger, so it has more overall spread.
  • The longer right whisker suggests the data is skewed to the right.
Self-check before you finish
  • Did I order the data first?
  • Did I use the correct lower and upper halves?
  • Did I separate median from quartiles clearly?
  • Did I explain what the numbers mean, not just list them?
