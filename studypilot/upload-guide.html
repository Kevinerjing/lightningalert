What to send

Best document types for analysis

PDF

Best for course calendars, worksheets, schedules, rubrics, and teacher notes.

PNG or JPG

Best for photographed homework, handwritten notes, and mistake notebook pages.

DOCX

Good for typed study guides or editable school documents.

TXT or pasted text

Fastest option for short notes, assignment instructions, or copied summaries.

How to share

Two easy ways to give a document to Codex

1. Upload it in chat

Drag the file into the chat or use the attach button, then say what you want done.

Example: Analyze this PDF and add the important test dates to the science page.

2. Send the full file path

Tell Codex exactly where the file is stored on your computer.

Example path: c:\Users\guanh\Downloads\science_schedule.pdf

Where to keep files inside Study Mission Assistant

Best workflow

Simple steps

1. Save the file into the right folder.

2. Upload it in chat or send the full path.

3. Tell Codex what you want: summarize, extract tasks, update subject pages, or analyze mistakes.

4. Codex can then update the app files directly.

Mistake photos

First version of mistake image intake

1. Put a mistake photo into uploads/mistakes/.

2. Run the update task.

3. Open the Mistakes Notebook page.

4. Look for the new draft card under Photographed mistake drafts.

5. Use the image path and suggested subject/topic to manually add the full mistake entry.
